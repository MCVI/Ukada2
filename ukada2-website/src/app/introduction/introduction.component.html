大赛简介

华中师范大学ACM程序设计竞赛是一年一度的面向全武汉各高校选手的编程竞赛。

本次比赛由华中师范大学计算机学院主办，华中师范大学ACM协会承办，旨在加强武汉地区各高校ACM选手的交流，分享优秀经验，吸引更多的编程爱好者参与。

比赛规则

于华中武汉地区就读的本科生均报名参加本赛事。 赛制为组队赛，每个队伍最多有三名成员，在比赛期间我们为每个队伍提供一台没有接入互联网的计算机。
比赛题目为 12 道，以中文描述。


选手可以携带任何印刷品入场以供比赛期间查阅。
选手不可携带任何电子设备进入赛场。比赛期间任何使用电子设备的企图将被视为作弊行为，并使选手失去参赛资格。
比赛期间提供代码打印的服务。
选手应无条件服从裁判的指令和判决。


选手所用计算机运行 Windows 7，安装有以下开发工具：
CodeBlocks
测试机的操作系统为 Ubuntu Linux，可用的编译器有：
GCC 5.2.1
Open JDK 7
比赛使用 DOM Judge 处理选手的提交。选手的程序只能通过标准输入输出与操作系统交互。


C 程序的编译命令为
$ gcc -std=c11 -O2 -o main main.cpp。
64 位整数的格式说明符为 %lld 和 %llu。

C++ 程序的编译命令为
$ g++ -std=c++11 -O2 -o main main.cpp。
64 位整数的格式说明符为 %lld 和 %llu。
